# Domain Routes Note

Status: intentionally not published as live route truth.

Public YonerAI docs may describe API contracts and local smoke checks, but they
must not publish live route maps, operator hostnames, tunnel layouts, or
control-plane routing instructions.

Use public contract docs such as `docs/contracts/external-agent-api.md` for
reviewable API shape. Official routing and deployment ownership remain separate
private/runtime/control-plane decisions.
