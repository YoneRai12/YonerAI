# 2026.06.01 - YonerAI CLI Local Runtime v0.6.4 Installer Execute Fix

Tag: `v0.6.4`

## Summary

v0.6.4 is a stable patch for the YonerAI CLI Local Runtime installer. It fixes
the v0.6.3 PowerShell `install.ps1 -Execute` path where manifest artifact
metadata could be confused with a script parameter, causing the installer to
stop before downloading the release ZIP. The short install command remains
`irm https://install.yonerai.com | iex`.

## Installer fix

The v0.6.3 script could fail under PowerShell strict mode with an error like:

```text
The property 'url' cannot be found on this object.
```

v0.6.4 keeps the selected manifest artifact as structured metadata and uses a
separate internal name for it. A regression test now runs the execute path with a
local fixture ZIP, verifies artifact SHA256, and confirms the local bootstrap
handoff is reached.

## Source boundary

`yonerai.com/install` is command text and wrapper entry only. It must not serve
`install.ps1`, release manifests, ZIP artifacts, hash sidecars, or any local PC
file used for installation.

The install source of truth is GitHub Releases:

```text
https://github.com/YoneRai12/YonerAI/releases/latest/download/install.ps1
https://github.com/YoneRai12/YonerAI/releases/latest/download/install.ps1.sha256
https://github.com/YoneRai12/YonerAI/releases/download/v0.6.4/manifest.v0.6.4.json
https://github.com/YoneRai12/YonerAI/releases/download/v0.6.4/YonerAI-0.6.4.zip
```

## Security boundary

- Bootstrap execution can be hash-checked before running.
- The release ZIP SHA256 must match the manifest before extraction.
- Custom/local manifests and ZIP artifacts are rejected by `install.ps1`.
- Stable channel does not select alpha artifacts.
- Alpha still requires explicit `-Channel alpha`.
- No production signing key or production trust store is included.
- No PATH mutation, registry edit, service install, admin request, provider key
  storage, shared traffic, forced update, or auto-update apply is enabled.

## What changed since v0.6.3

- Fixed the `install.ps1 -Execute` artifact metadata bug.
- Added execute-path regression coverage for manifest artifact resolution and
  ZIP SHA256 verification.
- Updated stable install documentation and site content to `v0.6.4`.
- Kept `install.yonerai.com` as the short command entry and GitHub Releases as
  the distribution source.

## Known limitations

- This is not full YonerAI cloud production.
- This is not a production-signed installer.
- Auto-update/apply is not implemented.
- npm, winget, Microsoft Store, and production installer channels are not ready.
- Production cloud runtime and live integrations are not included.
- Production account login is not included.
- External providers remain opt-in and keys are not printed or stored by the
  installer.
