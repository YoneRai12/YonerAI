## Security and Hygiene

- The CLI only accepts loopback API origins by default.
- The CLI does not call external providers directly.
- The CLI does not run shell commands or deployment commands.
- The CLI does not persist memory.
- `ORA_CORE_API_TOKEN`, when configured, is sent to Core as a header and is not printed.
