## Highlights (JP)
- **Setup UIを全面刷新**:
  - 左サイドバー + 右カードの管理画面へ再構成
  - 言語切替（日本語/英語）
  - Quick Actions（Roles / Permissions / Approvals / Relay）
  - 4K含む解像度に応じたレスポンシブ最適化
- **外部APIを正式ルート化**:
  - `POST /api/v1/agent/run`
  - `GET /api/v1/agent/runs/{run_id}/events`
  - `POST /api/v1/agent/runs/{run_id}/results`
  - `ORA_WEB_API_TOKEN` 前提で保護
- **Run state 安定化**:
  - メモリ上のアクティブrun上限を env で制御 (`ORA_MAX_ACTIVE_RUNS`)
  - run state のTTL cleanup (`ORA_RUN_STATE_TTL_SEC`)
- **Cloudflare/Tunnel運用の実務改善**:
  - tunnel子プロセスの追跡・停止・stale pid掃除を追加
  - `ORA_TUNNELS_NEW_CONSOLE` でWindows表示挙動を切替
- **一時ダウンロードリンク改善**:
  - `DOWNLOAD_PUBLIC_BASE_URL` 対応
  - quick tunnel URL の自動再利用/補完を強化
- **音楽メンション処理の分離**:
  - mention音楽トリガー処理を専用ハンドラ化
  - playlist時の挙動を `ORA_MUSIC_MENTION_PLAYLIST_MODE` で切替
