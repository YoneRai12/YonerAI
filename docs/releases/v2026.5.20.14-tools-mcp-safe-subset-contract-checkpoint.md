# v2026.5.20.14 Tools/MCP Safe Subset Contract Checkpoint

Date: 2026-05-20
Status: public contract checkpoint. Not tools/MCP completion, not runtime execution, not deploy, and not production.

## Summary

This checkpoint defines the public-safe Tools/MCP contract boundary for the v7.7 public repo. It keeps Tools/MCP as a bounded capability surface and explicitly denies unrestricted runtime execution.

## Highlights

- Tools/MCP remains disabled by default in public-safe paths.
- Unknown or undeclared tool capabilities are denied.
- Shell execution, file writes, deploy actions, secret access, dynamic MCP discovery, and automatic code mutation remain outside the public MVP.
- Approval-gated action semantics and audit-event expectations are documented.

## User-visible changes

- README and the capability matrix point readers to this Tools/MCP safe subset checkpoint.
- Public-facing wording now presents Tools/MCP as a contract boundary, not as a completed runtime feature.
