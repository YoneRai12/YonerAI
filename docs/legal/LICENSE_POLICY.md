# YonerAI License Policy

Status: public distribution policy for the YonerAI public repository. This is a
licensing and distribution boundary document, not a production readiness claim.

## Summary

YonerAI is source-available and noncommercial by default. It is not OSI open
source.

| Surface | Default license policy |
| --- | --- |
| Code | PolyForm Noncommercial License 1.0.0 |
| Documentation | CC BY-NC-ND 4.0 unless a file says otherwise |
| Images, release cards, diagrams, screenshots, public copy | CC BY-NC-ND 4.0 unless a file says otherwise |
| YonerAI name, logo, product identity, domains, release names | All Rights Reserved |
| Third-party dependencies | Their own upstream licenses |

## Allowed by default

- Personal local use.
- Evaluation and research use.
- Noncommercial experimentation with the local CLI runtime.
- Internal review of public contracts, tests, release manifests, and docs.

## Not allowed without a separate commercial license

- Commercial use.
- Paid hosted service use.
- Resale or redistribution as a product or service.
- Sublicensing.
- Production business use.
- Use of the YonerAI name, logo, domain identity, or brand assets as a mark for
  another product or service.

## Commercial licensing

Commercial licensing is reserved. Use the public repository owner channel as the
contact placeholder until a dedicated licensing contact is published.

## No trademark license

The license for source code and documentation does not grant trademark rights.
The YonerAI name, logo, product identity, domains, release names, and related
brand assets remain All Rights Reserved.

## Distribution boundary

The public repository may contain release manifests, install guidance, local CLI
runtime code, tests, and public contracts. It must not contain production
signing keys, production trust stores, production Oracle/control-plane runtime,
deploy secrets, live Discord tokens, or private operational inventory.

## Third-party notices

Third-party package metadata may include MIT, Apache, BSD, ISC, MPL, LGPL,
Python, CC0, CC-BY, or other licenses. Those apply to the third-party packages.
They do not make YonerAI itself MIT-licensed or OSI open source.
