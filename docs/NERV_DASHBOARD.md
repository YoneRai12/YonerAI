# Retired Legacy Dashboard Note

The old NERV-style dashboard surface was tied to the retired `ora-ui/` tree.

It is no longer part of the active public MVP surface and must not be used as the product foundation for new public work. Current public user-visible work should use the Core API contracts and, when a web smoke/demo is needed, `clients/web/`.

This note is kept only as historical context. It is not a production dashboard claim, not a live telemetry claim, and not a raw chain-of-thought exposure claim.
