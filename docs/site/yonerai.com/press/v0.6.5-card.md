# YonerAI v0.6.5 release card

Share URL:

```text
https://yonerai.com/releases/v0.6.5
```

Title:

```text
YonerAI CLI Local Runtime v0.6.5
```

Short copy:

```text
YonerAI CLI Local Runtime repair-safe installer update.
Quick install is `irm https://install.yonerai.com | iex`, and existing or
partial installs now require explicit repair/retry choices before reinstalling.
```

Highlights:

- Repair-safe handling for existing or partial install targets
- `-Repair`, `-Force`, and `-CleanRetry` preserve the old target as backup
- PATH mutation remains off by default; `-SetPath` is explicit
- Quick install from a Cloudflare static wrapper
- GitHub Release remains the distribution source for executable assets
- Startup home header for `yonerai`
- Update check/plan shows channel, artifact, trust, and next safe command
- `yonerai.com` remains a command page, not an installer host

Boundary:

```text
Not production cloud. Not a production-signed installer. No live integrations,
production account login, npm, winget, or forced auto-update.
```
