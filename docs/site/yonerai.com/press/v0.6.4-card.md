Title:

```text
YonerAI CLI Local Runtime v0.6.4
```

Short copy:

```text
YonerAI CLI Local Runtime installer fix.
Quick install is `irm https://install.yonerai.com | iex`, and the wrapper
checks install.ps1.sha256 before running the GitHub Release bootstrap script.
```

Highlights:

- Fixes the v0.6.3 PowerShell execute-path artifact resolution bug
- Quick install from a Cloudflare static wrapper
- GitHub Release remains the distribution source for executable assets
- Verified install checks `install.ps1.sha256` before execution
- `install.ps1` verifies manifest and release ZIP SHA256 before install-like steps
- Update check/plan shows channel, artifact, trust, and next safe command
- `yonerai.com` remains a command page, not an installer host

Boundary:

```text
Not production cloud. Not a production-signed installer. No live integrations,
production account login, npm, winget, or forced auto-update.
```
