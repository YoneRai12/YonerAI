# YonerAI v0.11.0-alpha.1 - Account Sync and Oracle API Foundation

YonerAI adds a public-safe account sync foundation.

## Highlights

- Google auth dry-run contract
- cloud/local sync boundary
- `yonerai sync status`
- `yonerai sync preview`
- `/同期` in the Japanese Mission Control CLI
- Official API fixture for future yonerai.com / YonerAIOracle
- user/device/provider rate-limit policy shape
- private self-evolution backend handoff

## Share copy

YonerAI v0.11.0-alpha.1 adds the public account-sync and Official API contract
foundation: `/同期`, `yonerai sync status`, sync preview/approval dry-run, rate
limit policy shape, and a sanitized private YonerAIOracle handoff.

Not production cloud. No production Oracle, no production Google login, no live
Discord, no Official Managed Cloud runtime in the public repo, no OpenAI shared
traffic runtime, and no automatic local-to-cloud upload.
