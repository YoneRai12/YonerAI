# v0.5.0 release card content

Status: copy source for a future manually prepared social/release card. This is
not a deployment artifact and not a production launch claim.

## Card title

YonerAI CLI Local Runtime v0.5.0

## Card subtitle

Install the local CLI, type `yonerai`, choose a language, and start from the
Mission Control terminal.

## Short copy

YonerAI v0.5.0 is stable for the local CLI runtime slice. It includes the
interactive Mission Control CLI, Japanese-first settings, mock provider by
default, loopback-only local LLM guidance, task progress, reviewer/subagent
planning, redacted run history, dry-run installer manifest planning, and a
plan-first local bootstrap helper for archives/checkouts that include
`install-local.ps1`.

Local CLI runtime only. Not production Oracle, not Official Managed Cloud
runtime, not live Discord, and not a network-executing installer.
