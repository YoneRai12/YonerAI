## Title

YonerAI CLI Local Runtime v0.5.1 Distribution Trust Update

## Summary

v0.5.1 is a stable patch release for the local CLI runtime distribution path.
It keeps the v0.5.0 Mission Control CLI and adds release trust hardening:
`install-local.ps1`, a v0.5.1 manifest, deterministic source archive generation,
and clearer source-available noncommercial licensing.

## Primary command

```powershell
yonerai
```

## Local bootstrap command block

```powershell
cd "$HOME\Downloads\YonerAI-0.5.1"
.\install-local.ps1
.\install-local.ps1 -Execute -Launch
```

The first command prints the plan only. The second command is an explicit local
install into `.venv`; it does not mutate PATH, run a remote script, request
admin rights, install a service, or enable live providers.

## Manifest commands

```powershell
yonerai manifest verify releases/manifest.v0.5.1.json --pretty
yonerai install plan --manifest releases/manifest.v0.5.1.json --pretty
yonerai update plan --manifest releases/manifest.v0.5.1.json --pretty
```
