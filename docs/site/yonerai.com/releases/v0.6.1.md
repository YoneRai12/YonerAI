# YonerAI CLI Local Runtime v0.6.1

Status: stable local CLI runtime patch release content for
`https://yonerai.com/releases/v0.6.1`.

## Install

```powershell
& ([scriptblock]::Create((irm https://github.com/YoneRai12/YonerAI/releases/latest/download/install.ps1))) -Execute -Launch
```

This command gets installable bytes from GitHub Release assets, not from
`yonerai.com`.

## Source boundary

- `yonerai.com`: install guide and command surface only.
- GitHub Releases: `install.ps1`, `install.ps1.sha256`, manifest, and ZIP
  assets.
- Local PC files: not used as installer source.

## GitHub Release

https://github.com/YoneRai12/YonerAI/releases/tag/v0.6.1

## Not included

No production cloud runtime, no production Oracle, no live Discord, no
production Google login, no npm/winget channel, and no production signing/trust
store.
