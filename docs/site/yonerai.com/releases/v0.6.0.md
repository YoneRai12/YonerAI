## Title

YonerAI CLI Local Runtime v0.6.0

## Summary

v0.6.0 is a stable release for the local CLI runtime slice. It promotes the
v0.6 terminal UI from alpha and adds post-alpha hardening: Quality Wall CI,
auth/privacy dry-run visibility, safer update/install status, local ledger
hardening, and a clearer Japanese-first interactive screen.

This is not full YonerAI cloud production and not a production network
installer.

## Manifest commands

```powershell
# Release ZIP flow: download manifest.v0.6.0.json as a separate asset first.
$manifest = ".\manifest.v0.6.0.json"
yonerai manifest verify $manifest --pretty
yonerai install plan --manifest $manifest --pretty
yonerai update check --manifest $manifest --pretty
yonerai update plan --manifest $manifest --pretty
```

These commands read local files and print verification or dry-run plans. They
do not download, install, mutate PATH, execute remote scripts, request admin,
write services, or connect to a production control plane.

The generated ZIP does not embed `releases/manifest.v0.6.0.json`; the manifest
is published as its own GitHub Release asset so it can record the ZIP hash.
Users installing from ZIP should download `manifest.v0.6.0.json` next to the
extracted folder or run these commands from a repository checkout.
In a checkout, set `$manifest = "releases\manifest.v0.6.0.json"` instead.
