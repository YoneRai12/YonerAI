# YonerAI site content contracts

Status: content foundation for future yonerai.com pages. This directory is not
a deployment target and contains no Cloudflare, Vercel, DNS, tunnel, analytics,
or secret configuration.

Current content contracts:

- `yonerai.com/install.md` maps to future `https://yonerai.com/install`.
- `yonerai.com/releases/v0.5.0.md` maps to future
  `https://yonerai.com/releases/v0.5.0`.
- `yonerai.com/press/v0.5.0-card.md` is copy source for a manually prepared
  release card or social card.

GitHub Releases remain the distribution source of truth. yonerai.com content is
the future user-facing install/share surface.
