# 🚀 ORA Bot 究極の「1ステップ」移行ガイド (Windows ↔ M4 Mac)

このガイドでは、新開発の自動ランチャーを使用して、Windows PC から M4 Mac へ ORA Bot を「ダブルクリックするだけ」で移設する手順を説明します。

---

## Step 1: Windows 側でパック作成

既存のフォルダで以下のコマンドを 1 回だけ実行します。

```powershell
# Windows のターミナル (PowerShell等) で実行
python migration_helper.py pack
```

*   **何が起きるか**: 移行に必要な全ファイルを `ora_migration.zip` という 1 つのファイルにまとめ、同時に **Mac 用の起動ボタン** を作成して中に入れます。
*   このファイルを USB、AirDrop (Mac側で受信)、Google ドライブ等を使って、**M4 Mac の好きな場所にコピー**してください。

---

## Step 2: Mac 側でダブルクリック

コピーした Zip ファイルを Mac で右クリック -> 展開します。
出てきたフォルダの中にある、以下のファイルを**ダブルクリック**してください。

📄 **`Double_Click_To_Start.command`**

**これだけで以下の全てが自動で行われます:**
1.  **AIの脳の準備**: Python のセットアップと AI ライブラリの自動インストール。
2.  **画面の準備**: 管理用画面 (ora-ui) の依存関係インストールと起動。
3.  **Bot の起動**: 準備が整い次第、自動的に ORA Bot が立ち上がります。

---

## (参考) 最初の 1 回だけ必要な設定
Mac で初めて起動する場合、以下の 2 点だけ確認してください。

1.  **ファイル実行の許可**: ダブルクリック時に「開発元を確認できない」と出た場合は、そのファイルを右クリック ->「開く」を選択してください。
2.  **設定ファイルの確認**: 自動的に作られた `.env` ファイルに、あなたの Discord トークンを入力してください。

これで移行は完了です！コマンド入力は一切不要になりました。
もしエラーが出る場合は、いつでもお声がけください。

---

## (応用) Windows からコードを編集したい場合

「Windows の VS Code で、Mac 上のコードを直接編集したい」場合は、以下の設定を Mac で行ってください。

1.  **Macの設定**:
    *   システム設定 -> 一般 -> 共有 -> **「リモートログイン」をON** にする。
    *   (これだけで、Windows から SSH で接続できるようになります)

2.  **Windows (VS Code) の設定**:
    *   拡張機能 **「Remote - SSH」** をインストール。
    *   左下の緑のボタン -> `Connect to Host` -> `ssh user@mac-ip-address`
    *   (Macのユーザー名とパスワードを入力)

これで、**まるで Windows にあるかのように** Mac 上のファイルを編集できます。
